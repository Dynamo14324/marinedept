# Executive Management Dashboard Design: Marine Department Audit Analysis

## 1. Executive Summary
This dashboard is designed for **Executive Management** to provide a comprehensive, interactive overview of the **Marine Department's** audit performance and compliance status. It replicates and enhances the visualizations found in the 2024 Management Review, focusing on key performance indicators (KPIs) for **HSQE (Health, Safety, Quality, and Environment)**.

## 2. Dashboard Architecture
The report is structured into four primary interactive pages:

| Page | Focus | Key Visuals |
| :--- | :--- | :--- |
| **Executive Overview** | High-level KPIs and Fleet Health | Total Audits, NC Rate, Fleet Comparison (Tanker vs. Dry), Vetting KPI Status. |
| **Audit Performance** | Historical Trends and Quarterly Analysis | 3-Year NC Trend, Quarterly Audit Distribution, Auditor Performance. |
| **Deficiency Analysis** | Root Cause and Category Breakdown | NCs by Category (e.g., Safety Management, Pollution Prevention), Vessel-specific issues. |
| **Compliance & Schedule** | Forward-looking Planning | IA Planner status, Expiring Audit alerts, Travel & Onboard days summary. |

## 3. Key Performance Indicators (KPIs)
- **NC per Audit:** Target < 3.0 (Vetting) and < 0.10 (ISM).
- **Audit Completion Rate:** Percentage of planned audits executed.
- **Fleet Performance Index:** Comparative score between Tanker and Dry fleets.

## 4. Data Integration
Data has been consolidated from multiple years (2023-2025) and various sources:
- **Internal Audit Logs:** Detailed records of NCs and Observations.
- **Safety Inspections:** Quarterly performance tracking.
- **Deficiency Categories:** Granular analysis of findings across 60+ categories.
- **Visit Planner:** Real-time tracking of superintendent visits and vessel status.

## 5. Interactive Features
- **Dynamic Slicers:** Filter by Year, Fleet Type, Auditor, and Vessel Name.
- **Drill-through:** Click on a category to see specific vessel findings.
- **Trend Analysis:** Toggle between monthly, quarterly, and yearly views.
- **Executive Summary Tooltips:** Hover over charts for detailed management notes.
